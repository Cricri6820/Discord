Comment créer un serveur?

  1. Cliquer sur le bouton "+"
  2. Cliquer sur Créer un serveur
  3. Créer des rôles
  4. Créer des salons
  5. Inviter quelques bots
  6. Remplir les salons d'infos (règlement, FAQ, info rôle, ...)

Salons indispensables

  • Salon de bienvenue
  • Le règlements, c'est très important
  • Un salon d'annonces
  • Un salon dans lequel les membres peuvent écrire
  • Un salon pour les commandes de bots

Les permissions

  • L'administrateurs ont presque tous les permissions
  • Gérer le salon : Il peuvent renommer, changer les permissions et supprimer le salon
  • Changer le nom d'utilisateur : Permet de changer son pseudo visible que sur le serveur
  • Utiliser un emoji exterme : Permet les utilisateur de Nitro d'utiliser n'importe quelle emoji persos

Bots suggérés

  • MEE6
  • Dyno
  • UnbelievaBoat
  • Giveaway Bot
  • Zira

N'hésitez pas à activer le serveur communautaire en créant un serveur

  • Pouvoir de créer des salons d'annonces
  • Ajouter un ecran de bienvenue
  • Voir les statistiques du serveur

Prérequis de serveur communautaire

  • Avoir un salon de règlements et un pour les staffs
  • Activer un niveau de sécutité du serveur
