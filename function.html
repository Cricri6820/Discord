Fonctionnalités de base

Comme tous les utilisateurs connaissent Discord, on peut envoyer des messages avec les autres

Vous pouvez faire un retour à la ligne avec Shift+Enter

Les utilisateurs peuvent envoyer la demande d'ami ou bloquer les autres

Type de salons

Il y a 4 type de salons sur Discord

  • Textuelle: Pour envoyer des messages
  • Vocal: Pour parler avec les autres ou les écouter
  • Annonce: C'est le salon là où les administrateurs peut envoyer des annonces, aussi les membres peuvent les suivre depuis leur serveur, prérequis: Activer le serveur Communautaire
  • Boutique: C'est le salon là où les membre peuvent acheter un jeu, prérequis: Avoir acheté le passe de developpeur

Nitro
